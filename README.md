# Портфолио исследовательских проектов

## Проект 1: Прогнозирование временных рядов
**Папка:** `Forecasting Time Series/`

Прогнозирование индекса потребительских цен (ИПЦ) в России. Сравнение традиционных эконометрических моделей (VAR, ARIMA) с ML-алгоритмами (Random Forest, XGBoost, SVM, MLP).

**Результат:** Простые модели не уступают сложным. MLP показал худшие результаты из-за переобучения.

**Файлы:** `Forecasting time series.ipynb` (код), `Forecasting time series.pdf` (текст)


## Проект 2: Инфляционные ожидания и новостной фон
**Папка:** `Kalman Filter and Rational Inattention/`

Моделирование инфляционных ожиданий домохозяйств через линейный фильтр Калмана (рациональная невнимательность). Построение новостных индексов (тематическое моделирование ARTM + сентимент-анализ SiBERT) для объяснения разрыва между модельными и опросными ожиданиями (множественная регрессия, оцененная 2SLS).

**Результат:** Домохозяйства со сбережениями реагируют на широкий круг новостей, в основном, макроэкономических, без сбережений — преимущественно на фискальные.

**Файлы:** `kalman_filter_analysis.ipynb` (код), `thesis_summary.pdf` (краткое изложение), `thesis_full.pdf` (полный текст), `presentation.pdf` (презентация)


## Проект 3: Фискальная теория уровня цен
**Папка:** `Fiscal Theory of the Price Level/`

DSGE-модель взаимодействия фискальной и монетарной политики (Leeper, 1991). Анализ двух режимов и чувствительности к параметру α.

**Результат:** Единственное устойчивое равновесие при -2 < α < 0.

**Файлы:** `part_d/` (код Dynare для части d), `part_e/` (код Dynare для части e), `Fiscal Theory of the Price Level_analysis.pdf` (полный анализ)
